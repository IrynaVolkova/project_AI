This repository contains my solutions for Harvard CS50's Introduction to Artificial Intelligence with Python.  
It demonstrates hands-on experience with search algorithms, game AI, and Python programming

Each project has its own folder:

attention (implementing the attention mechanism in Python)
crossword (solving crossword puzzles using CSPs (Constraint Satisfaction Problems))
degrees (graph traversal for social network connections)
heredity (probability and genetics problems)
knights (backtracking algorithms)
minesweeper (AI to play Minesweeper)
nim (game strategy for Nim)
pagerank (implementing the PageRank algorithm)
parser (simple language parser)
shopping (constraint satisfaction for shopping problem)
tictactoe (AI player for Tic Tac Toe)
traffic (traffic simulation and optimization)
